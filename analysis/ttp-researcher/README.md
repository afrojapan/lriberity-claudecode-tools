# TTPリサーチャー GAS版

ベンチマーク動画（バズっている参考動画）のURLを入れると、**YouTubeのアルゴリズムが関連として紐付けているチャンネルを自動で抽出**してくれるツールです。

Google スプレッドシート + GAS（Google Apps Script）で動くため、**インストール不要・Googleアカウントがあれば誰でも使えます。**

---

## 事前準備

スプレッドシートのテンプレートを自分のドライブにコピーしてください。

---

## 基本的な使い方（6ステップ）

### STEP 1 — ベンチマーク動画URLを入力する

「設定」シートを開き、参考にしたい動画のURLを最大10件入力します。

**複数入力が強力です。** 2〜5本入れると「複数動画に共通して出てくるチャンネル」が可視化され、アルゴリズムに乗っているチャンネルを見抜けます。

### STEP 2 — フィルタ条件を設定する

| 項目 | 説明 |
|---|---|
| 最小再生数 | これ未満の動画を除外 |
| 最大登録者数 | 大手チャンネルを除くのに便利 |
| 最小登録者数 | 小規模すぎるチャンネルを除外 |
| 1動画あたりの取得件数 | 増やすほど網羅性が上がるが時間がかかる |
| 動画公開から何日以内 | 古い動画を除くのに便利 |
| チャンネル開設日以降 | 新興チャンネルに絞り込む場合に使用 |
| 直近投稿から何日以内 | 休止チャンネルを除くのに便利 |

### STEP 3 — リサーチを実行する

メニュー「**TTPリサーチ**」→「**▶ リサーチ開始**」をクリック。1〜3分で完了します。

### STEP 4 — 結果シートを確認する

「結果_MM-dd_HHmm」という名前のシートが自動生成されます。

| 列 | 意味 |
|---|---|
| **出現ベンチマーク** | ①②③形式で表示。多いほどアルゴリズムに乗りやすいチャンネル |
| **ヒット動画数** | フィルタを通過した動画の本数（チャンネル単位） |
| **平均バズ率** | 再生数 ÷ 登録者数の平均。高いほど伸びている動画がある |
| **開設日 / 最終投稿日** | チャンネルの活動状況を判断するのに使う |
| **サムネイル** | 最もバズ率の高い「代表動画」のサムネイル画像 |

**順位の色の意味：**
- 🥇 金 = 1位（最優先候補）
- 🥈 銀 = 2位
- 🥉 銅 = 3位

### STEP 5 — 気になるチャンネルに「転記」チェックを入れる

左端の「転記」列のチェックボックスをONにします。

### STEP 6 — リサーチシートに転記する

メニュー「**TTPリサーチ**」→「**🔬 リサーチシートに転記**」を実行。

選択したチャンネルの詳細情報（アイコン・バナー・トップ5動画）が自動転記されます。

---

## 除外リストの活用

「除外リスト」シートにチャンネルIDを追加しておくと、次回リサーチ時に結果から自動除外されます。

- A列：チャンネルID（`UC〜` 形式）
- B列：メモ（任意）

---

## 実践ワークフロー

```
① バズっている競合動画を3〜5本選ぶ
      ↓
② 「設定」シートに貼り付け
      ↓
③ TTPリサーチ → リサーチ開始
      ↓
④ 金・銀・銅の行を優先してチェック
      ↓
⑤ リサーチシートに転記 → チャンネル深掘り
      ↓
⑥ TTP実行 or 除外リストに追加
```
